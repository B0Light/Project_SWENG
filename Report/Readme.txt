* 실행방법 1 - 웹 빌드 버전에서 프로그램 실행

1. 웹 빌드 주소 : https://sodaxi.itch.io/tile-adventure
- 해당 사이트에 접속하면 웹 환경에서 본 프로그램을 실행할 수 있다.
- Run game 버튼을 눌러 실행할 수 있다.

* 실행방법 2 - 유니티 에디터에서 프로그램 실행

1. Unity hub 다운로드 : https://unity.com/kr/download
- 해당 사이트에서 Unity hub를 다운로드 받아 설치한다.

2. Unity hub를 실행하여 로그인 한 후 Installs 탭에서 Install Editor를 클릭하여 2021.3.30f1 버전을 다운로드 받는다.
- 만약 Install Editor에서 보이지 않는다면 유니티 다운로드 아카이브 : https://unity.com/releases/editor/archive
에서 해당 버전을 찾아 설치한다.

3. 산출물을 관리하고 있는 GitHub 주소 : https://github.com/SodaXI/Project_SWENG
- 해당 GitHub에서 Project_SWENG 폴더를 다운받는다.

4. Unity hub의 Projects 탭에서 Open 버튼을 눌러 다운받은 Project_SWENG 폴더를 선택한다.

5. Projects에 추가된 Project_SWENG 프로젝트를 실행하면 해당 프로젝트 작업 화면이 표시되고, 중앙 상단의 재생, 일시 정지, 스텝 버튼 중 재생 버튼을 눌러 프로그램을 실행한다.
- 해당 프로그램의 해상도는 Full HD(1920x1080) 환경에 맞추어져 있으므로 Game 뷰 화면의 해상도가 Full HD(1920x1080)인지 확인한다.
